# vite

## 依赖安装

1. esbuild  打包工具 
2. cac      脚手架
3. es-module-lexer  import语句解析
4. fs-extra  文件处理
5. magic-string 字符串处理 （sourcemap
6. rollup 线上builder
7. picocolors  命令行输出颜色
8. connect     http中间件服务
9. resolve     node的路径解析算法 （包括node_modules)


## node自带
fs、path
```bash
pnpm i cac es-module-lexer esbuild fs-extra magic-string rollup -w
```
